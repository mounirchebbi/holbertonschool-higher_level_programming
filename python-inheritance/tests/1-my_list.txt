# 1-my_list.txt

How to use 1-my_list.py module
------------------------------

Defines MyList class that inhertits from list class

Instantiation
>>> MyList = __import__('1-my_list').MyList

>>> list1 = MyList()
>>> print(type(list1))
<class '1-my_list.MyList'>
>>> list1.print_sorted()
[]
>>> for i in range(4, 0, -1):
...     list1.append(i)
>>> print(list1)
[4, 3, 2, 1]

>>> list1.print_sorted()
[1, 2, 3, 4]

>>> list2 = MyList([3, 2, 1])
>>> list2.print_sorted()
[1, 2, 3]

>>> list2 = MyList(["C", "B", "A"])
>>> list2.print_sorted()
['A', 'B', 'C']

Expected Errors
---------------
>>> list1 = MyList(None)
Traceback (most recent call last):
TypeError: 'NoneType' object is not iterable
